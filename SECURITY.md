# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please **do not** create a public issue.

Instead:

1. **GitHub Security Advisory** (Recommended):
   - Visit https://github.com/valksor/go-toolkit/security/advisories
   - Click "Report a vulnerability"

2. **Email**: packages@valksor.com with "SECURITY: go-toolkit" in subject

## Response Timeline

- **Initial response**: Within 48 hours
- **Assessment**: Within 7 days
- **Patch**: Based on severity, typically within 14 days

## Supported Versions

| Version | Support Status |
|---------|----------------|
| 0.x     | Supported      |

Thank you for helping keep go-toolkit secure!
