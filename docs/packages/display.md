# display

Color and formatting utilities for terminal output.

## Description

The `display` package provides utilities for adding color and formatting to terminal output.

**Key Features:**
- Terminal color support
- Formatting utilities
- Cross-platform compatibility

## Installation

```go
import "github.com/valksor/go-toolkit/display"
```

## Usage

The `display` package provides utilities for colored terminal output. Please refer to the source code for detailed usage examples.
